# HabitFlow LP - 架空習慣トラッカー

ミニマルデザインのランディングページ実装例。
20〜30代ビジネスパーソン向けに「毎日続けやすい」UIを目指しました。

## 特徴
- 黒背景 + グラデーションアクセント + 余白多めでクリーン
- スマホファースト設計（max-width: 500px）
- 機能：1日1習慣登録、ストリーク可視化、やさしいリマインダー

## 技術スタック
- HTML5 / CSS3（変数活用）
- スマホ対応レスポンシブ

## 開発背景
- ベース：Claudeで生成（10分）
- 自分カスタム：--accent1の色変更（#00d4ffに調整）、CTAホバーエフェクト追加など
- 理由：よりエネルギッシュでクリック誘導を強化

https://shinarufort-blip.github.io/habitflow-lp/ （GitHub Pages）

## ポートフォリオとしてのポイント
- スナック池（飲食店温かみ系）と差別化してクールSaaS系に挑戦
